# Papers

本目录保存从 Zotero 来源整理出的项目 canonical paper notes。

默认语言：中文。

每篇 note 至少包含：

- bibliographic source
- PDF/full-text completeness
- mechanism claim
- tensor semantics
- critical defaults
- known failure modes
- relevance to one model for multi-horizon
- relevance to future-aware architecture
- relevance to MoE-style conditional computation
- adopted details
- rejected details
