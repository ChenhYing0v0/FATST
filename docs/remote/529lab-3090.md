## 每次实验前

先查看 GPU 状态：

```bash
scripts/remote/check_529lab_3090_gpus.sh
```

如果 SSH alias 不同，显式传入 host：

```bash
scripts/remote/check_529lab_3090_gpus.sh user@host
```

## 选择 GPU

- 优先选择 memory used 更低的 GPU。
- 默认优先 GPU 1 或 GPU 2。
- 避免 GPU 0，除非用户明确批准。
- 不要把显存预算压满，给共享服务器留安全余量。

## 运行记录

每次 meaningful run 至少记录：

- local commit or dirty state
- remote host
- conda env
- selected GPU
- `nvidia-smi` snapshot
- exact command
- output path
- dataset and horizon settings
- seed settings
